  1. Load a video — drop a file (MP4, MOV, MKV, AVI) onto the app or click the drop zone to browse.
  2. Draw crop zones — click ✚ Draw new crop zone then drag on the source canvas. Create one for gameplay, one for HUD, minimap, facecam, etc.
  3. Position zones on the output — drag and resize zones on the 9:16 output canvas. Use the coordinate inputs on zone cards for pixel-perfect placement.
  4. Trim the timeline — drag the green in/out handles to export only the segment you want.
  5. Manage audio — click audio track chips below the timeline to mute/unmute individual tracks (game audio, mic, etc.).
  6. Save presets — click ★ Save layout as preset to reuse layouts across videos. Use the ↺ button to update existing presets.
  7. Export — hit Export 9:16 Video. FFmpeg renders locally and the file downloads automatically.
Tips
  • Toggle ⬡ outlines on the input or output panel header to show/hide zone borders on that canvas.
  • Use the 👁 toggle on a zone card to temporarily hide it from the output without deleting it.
  • Undo supports up to 50 steps of layout changes.
  • Change output resolution and FPS in the header controls.
